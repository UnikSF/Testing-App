# Matrix Planning

A school schedule management application for planning classes, assigning rooms, and managing teacher and student availability.

**Status: not yet started**

## What it Will Do

- Define classes (subject, teacher, student group, duration)
- Manage rooms (capacity, equipment, availability)
- Track teacher and student group availability / constraints
- Generate or manually build a weekly school timetable
- Detect conflicts (double-booking of rooms, teacher overlap, etc.)
